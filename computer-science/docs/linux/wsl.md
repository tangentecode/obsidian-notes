# Wsl

Integrate a **Linux** [shell](bash.md) in **Windows** (Windows Terminal)

## Installing WSL2

- Install via **Chocolatey**:

```shell
choco install wsl2
```

> !!Enable required features!!

## Installing Desired **distro**

### Ubuntu, SUSE, Debian, Kali â€¦

- Run `wsl --list --online`
- Install using `wsl.exe --install <Distro>`
