# Creating-project

- Use

```shell
cargo new project_name
```

- This will create:
```shell
project_name/
│
├── .gitignore  
├── Cargo.toml  # Main Flask application
├── src/               # HTML templates
│   └── main.rs 
├── .git/           # Static files (CSS, JS, images)
│   └── ...
├── requirements.txt  # Third-party libraries used   
│ 
```
