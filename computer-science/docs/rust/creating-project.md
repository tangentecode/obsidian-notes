# Creating-project

- Use

```shell
cargo new project_name
```

- This will create:
```shell
project_name/
│
├── Cargo.toml         # Main Flask application
├── src/               # HTML templates
│   ├── main.rs
│  
├── static/           # Static files (CSS, JS, images)
│   └── style.css
├── requirements.txt  # Third-party libraries used   
│ 
```
