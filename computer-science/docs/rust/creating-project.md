# Creating-project

- Use

```shell
cargo new project_name
```

- This will create:
```shell
project_name/
│
├── .gitignore         # Contains the ignored files for git
├── Cargo.toml         # Dependecies and their desired versions
├── src/               # HTML templates
│   └── main.rs 
├── .git/              # Static files (CSS, JS, images)
│   └── ...

```
