# Huffman-coding

Compresses and decodes any string to bits

## Example

**Input string:** "AABCBAD"

### 1. [[rle]] (Frequency Table)

| -------- | ------------- |
| A        | 3             |
| B        | 2             |
| C        | 1             |
|          |               |
