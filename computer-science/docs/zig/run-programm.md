# Run-programm

## Instructions

- Create file with the ending `.c`
- Compile the code:
	1. In cs50 environment with `make`
	2. On local computer with `gcc`

- Run the file:
	1. In cs50 environment (or generally Linux) with `./`
	2. If you use `gcc` (MingGW) compiled file is usually called `a.exe` 

## Example

```bash
code hello.c

make hello

./hello
```
